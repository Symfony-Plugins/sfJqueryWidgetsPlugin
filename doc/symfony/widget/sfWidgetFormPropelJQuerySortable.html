Available options:

  • model: The model class (required)
  • url: The url of the action(routing, or module/action) to be called using Ajax (required)
  • list_id: The HTML id of the generated list (model_name_sortable by default)
  • param: Name of the parameter sent to the action specified in the url option (id by default)
  • key_method: Method used to retrieve the value of the list item (getId by default)
  • list_type: Type of the list either ul or ol (ul by default)
  • criteria: A criteria to use when retrieving objects (null by default)
  • start: Function that gets called when sorting starts (before sending the Ajax request)
  • success: A function to be called if the request succeeds
  • complete: A function to be called when the request finishes (after success and error callbacks are executed)
  • error: A function to be called if the request fails
  • beforeSend: A pre-callback to modify the XMLHttpRequest object before it is sent
  • peer_method: The peer method to use to fetch objects
  • connection: The Propel connection to use (null by default)
  • error: A function to be called if the request fails
